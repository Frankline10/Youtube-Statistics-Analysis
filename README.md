# YouTube Analytics Power BI Dashboard

This project is a fully interactive **Power BI dashboard** built using a dataset of global YouTube channels.  
The dashboard provides insights into **top-performing channels, categories, revenue distribution, and audience engagement trends**.

---

## Project Overview
The purpose of this dashboard is to analyze YouTube channel metrics such as:

- Subscriber count  
- Total views  
- Total uploads  
- Estimated earnings  
- Category-wise performance  
- Country-wise distribution  

Using these metrics, the dashboard highlights key patterns and allows users to explore top channels across multiple dimensions.
